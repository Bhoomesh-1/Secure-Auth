# 🚀 How to Start Frontend

## Simple Steps

### Step 1: Open a NEW Terminal/PowerShell Window

**Important:** Open a **NEW** terminal window (keep the backend terminal running!)

### Step 2: Navigate to Frontend Folder

```powershell
cd "c:\Users\GOUDA BHOOMESH\capstone project\frontend"
```

### Step 3: Start the Frontend

```powershell
npm start
```

---

## ✅ What Will Happen

1. **Frontend will compile** (takes 10-30 seconds)
2. **Browser opens automatically** to `http://localhost:3000`
3. **You'll see the Home Page** with navigation

---

## 🎯 Expected Output

You should see:
```
Compiled successfully!

You can now view secure-auth-frontend in the browser.

  Local:            http://localhost:3000
  On Your Network:  http://192.168.x.x:3000
```

## 🐛 Troubleshooting

### If you see "npm is not recognized":
- Make sure Node.js is installed
- Download from: https://nodejs.org/

### If you see module errors:
```powershell
cd "c:\Users\GOUDA BHOOMESH\capstone project\frontend"
npm install
npm start
```

### If port 3000 is busy:
- The terminal will ask if you want to use a different port
- Type `Y` and press Enter
